Noise Filtering


Objectives:
At the end of this practical you should be able to
- use rank filtering for noise reduction.
- different types of noises

Terminology:


See:
Median Filtering

Run the matlab "demos" and select the "Removing Noise from Image" from the "User's Guide" (under "Analyzing and Enhancing Images").

Now run the "Removing Noise By Median Filtering" demo. Apply the available image noise and then observe the effects of the noise reduction algorithms.

Try to find out;
What is the best filter for "gaussian" random noise?
What is the best filter for "salt and pepper"(ie:"drop-out")?
